# readSetglobals

Reads all setglobals given in a GAMS code and returns them.

## Usage

``` r
readSetglobals(file)
```

## Arguments

- file:

  A gams file or a vector containing GAMS code.

## Value

A vector of values the setglobal variables are set to with setglobal
variables as names.

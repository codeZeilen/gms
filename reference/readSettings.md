# readSettings

Reads all settings (parameters, globals, and their respective values) in
the given GAMS code.

## Usage

``` r
readSettings(file)
```

## Arguments

- file:

  A gams file or a vector containing GAMS code.

## Value

A vector of parameter values and their names.
